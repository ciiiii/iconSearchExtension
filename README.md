## 🔧 Usage

- 解压插件压缩包
- chrome 地址栏输入 `chrome://extensions/`， 打开 Chrome 浏览器的扩展程序管理界面，并在该界面的右上方的开发者模式按钮上打勾
- 在勾选开发者模式选项以后，页面上就会多出一些按钮，点击“加载正在开发的扩展程序(Load unpacked)”按钮，并选择刚刚解压的 Chrome 插件文件夹的位置
- 插件就安装好了，请不要删除或修改插件文件夹
